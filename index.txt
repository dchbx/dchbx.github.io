        <h3>
          <a class="anchor" href="#welcome-to-github-pages"><span class="octicon octicon-link"></span></a>Welcome to the Affordable Care Act API
          </h3>

        <p>The Affordable Care Act API (ACApi) is a flexible IT framework that enables agencies to rapidly establish and extend cost-effective, online health insurance marketplaces for individuals and small businesses.</p>

        <p>The ACApi's reusable components support core functions, promote interoperability and improve performance to help reduce the cost of implementing and operating an HBX.  These components include:</p>
        <ul>
          <li>architecture</li>
          <li>information model</li>
          <li>canononical vocabulary</li>
          <li>interface</li>
          <li>resources</li>
        </ul>

        <p>The DCHBX is using the ACApi to improve DCHealthLink operation and efficiency.  It also incorporates knowledge gained and lessons learned in a useful form for the benefit and reuse of others.</p>

        <p>The ACApi offers a flexible framework upon which to build health insurance marketplace technology. The design encourages reuse and extension by applying modern, proven system development principles:</p>

<ul>
  <li>Convention over configuration</li>
  <li>Open source tools and systems</li>
  <li>Test-first development</li>
  <li>RESTful Web services</li>
</ul>

<p>Scope: Covers individual and SHOP marketplaces.</p>>


        <p>The District of Columbia Health Benefit Exchange Authority (DCHBX) launched the ACApi project following a condensed development cycle, culminating in an October 1, 2013 go-live of the DCHealthLink site.</p>
